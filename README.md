# Voice Assistant (Speech Controlled App Launcher)

A Python voice-controlled assistant that listens to spoken commands, processes them, and performs actions on your computer.

# Features
- Records speech from microphone
- Converts speech to text using Google Speech Recognition
- Uses text-to-speech to respond
- Tells the current time
- Greets the user based on time of day
- Opens applications such as:
  - Visual Studio Code
  - You can personally alter app_paths to add any app name with a filepath to the executable
- Runs continuously until the user says "goodbye"
